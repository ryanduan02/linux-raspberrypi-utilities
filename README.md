# Linux Command Recipes for Raspberry Pi (and General Linux)

A curated set of Linux command “recipes” used for Raspberry Pi development. Most commands work on any Debian/Ubuntu-like system; Raspberry Pi–specific commands are clearly labeled.


# Compatibility

Primary target: Raspberry Pi OS (Debian-based)

Also works on: Debian/Ubuntu and similar distributions

Pi-specific commands are marked with (Raspberry Pi only).
